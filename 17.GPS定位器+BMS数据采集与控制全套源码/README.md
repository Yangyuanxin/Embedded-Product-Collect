# GPSBMS
本项目是深圳市谷米万物科技有限公司自主研发的GS10系列GPS定位器+BMS管理全套原理图、PCB、BOM（在/doc/硬件设计目录下）
以及应用层源代码等，采用BSD许可协议。任何企业和个人都可以下载、使用或者修改后使用。
